# README

Gem use:

* Trix editor - WYSIWYG editor for form

* Carrierwave - uploader for image

* Bootstrap4 - framework for stylization

* Kaminari - pagination

How start only local

* ruby must be 2.4.2

* git clone https://github.com/Yaroslav999/blogm.git

* bundle install

* rake db:migrate

* rails s
